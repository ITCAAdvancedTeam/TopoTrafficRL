# TopoTrafficRL

Implementation of reinforcement learning based dense traffic awareness method.

## Install

1. Install TopoTrafficRL
    ```sh
    pip3 install .
    ```

## Usage

1. Run the example
    ```sh
    python3 scripts/example.py
    ```
